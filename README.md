# Free Fire Emotes Gallery

A beautiful gallery of Free Fire emotes with search functionality and team configuration features.

## Features

- Browse all Free Fire emotes
- Search functionality to find specific emotes
- Team configuration form
- Responsive design with particle background effects
- Copy buttons for easy sharing

## Deployment

This project is ready to be deployed on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.

### Manual Deployment Steps

1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in your project directory
3. Follow the prompts to deploy

### Automated Deployment Scripts

For convenience, we've included deployment scripts:

- On Windows: Run `deploy.ps1` in PowerShell
- On Unix-like systems: Run `deploy.sh`

Or simply click the button below to deploy directly to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/your-username/emote-wesitw)

## Local Development

To run this project locally:

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
